Abstract:

Este quarto trabalho de casa teve como objetivo criar um servidor em NodeJS com ExpressJS, que permita representar a informação sobre um arquivo musical armazenada numa base de dados em GraphDB.

Sendo assim foram criadas duas rotas apenas, que fornecem as informações que são necessárias às duas páginas html do servidor. A página de índice exibe a listagem das obras, onde existe uma hiperligação para a página individual de cada obra, que contém as suas respetivas informações. Este servidor utiliza Pug como gerador das páginas html.

Foram utilizadas três queries em SPARQL que são utilizadas nas rotas do servidor para, listar todas as obras, obter a informação de cada obra, e por último para obter a lista das partituras de cada obra.
