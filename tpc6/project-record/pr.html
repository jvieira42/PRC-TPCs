Abstract:

Este sexto trabalho de casa teve como objetivo a realização de uma ficha sobre Web Semântica e as tecnologias a ela associadas. Nesta ficha foram propostos quatro exercícios.

No primeiro exercício, para realizar o grafo, importou-se a ontologia após corrigida no GraphDB e através da opção "Visual Graph" foi possível visualizar o grafo correspondente.

No segundo exercício, foi utilizado o programa protégé para criar uma ontologia para uma rede de supermercados. Essa ontologia foi posteriormente carregada no GraphDB de modo a realizar as queries propostas sobre esta.

No terceiro exercício, apenas foi proposta a realização de 5 queries sobre a base de dados da DBPedia.

Por último, foi proposta a realização de 6 queries sobre uma ontologia de pókemons, sendo esta carregada no GraphDB para fazer as queries.
