Abstract:

Este quinto trabalho de casa teve como objetivo apenas criar um dataset, construído com uma query em SPARQL. Esta query recolhe campos, sobre nadadores armazenados na dbpedia.org, tais como o nome, sexo e data de nascimento, tendo também como campos opcionais o local de nascimento, a data da morte e o abstract. Estes dados são armazenados num ficheiro em formato JSON.
