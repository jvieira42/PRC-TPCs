Abstract:

Este oitavo trabalho de casa teve como objetivo a realização de um servidor em NodeJS com Express, e respetiva implementação de uma API sobre a ontologia de cinema utilizada na aula. Esta ontologia está armazenada no GraphDB.

Sendo assim, foram feitos 3 roteadores diferentes: Filmes, Atores e Personagens. Cada um destes roteadores tem 2 rotas, uma que lista todas as entradas e outra, que para uma entidade específica retorna as informações sobre esta.
