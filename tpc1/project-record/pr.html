Abstract:

Este primeiro trabalho de casa tem como objetivo criar uma ontologia em sintaxe turtle de modo a representar a turma de PRC que está armazenada num ficheiro csv, obtido através do download da folha excel da turma.

Sendo assim foi utilizado o programa protégé para criar a ontologia com as classes UC, aluno e professor, e respetivos tipos de dados como nome, email, curso e id. Também foram criadas as relações entre estas entidades, como leciona e frequenta, assim como os inversos.

Tendo a ontologia criada, apenas foi necessário criar um pequeno parser em linguagem python que processasse o ficheiro csv e escrevesse para um ficheiro a que chamei individuals.ttl cada um dos alunos, na sintaxe pretendida para a ontologia que é turtle.

Finalmente, após processada a informação, bastou copiar o conteúdo do novo ficheiro individuals.ttl, para o ficheiro original da ontologia, obtendo assim uma ontologia com todos os alunos da turma no ficheiro turma.ttl.
