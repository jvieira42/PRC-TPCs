Abstract:

Este sétimo trabalho de casa teve como objetivo a realização de uma ou várias queries em SPARQL, para recolher informação sobre filmes e atores. Essa informação posteriormente seria para ser convertida para a ontologia realizada na aula sobre cinema.

Apenas foi realizada uma query com informação da DBpedia e extraída em formato XML. De seguida foi criada uma stylesheet XSLT para transformar a informação relevante para formato ontológico.

Sendo a informação da DBpedia um bocado irregular, poderão haver alguns indíviduos que não estão bem formatos, devido também à quantidade enorme de caracteres ilegais nos nomes das diferentes entidades.
